## Get started


### Clone the repo
```shell
git clone git@github.com:angelodalisay/test04112019.git
cd test04112019
```

### NPM
Make sure to have the latest NPM version
```shell
npm install npm@latest -g
```

### Install Angular CLI
```shell
npm install -g @angular/cli
```
